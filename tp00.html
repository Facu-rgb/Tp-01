pregunta 6:que es html y en que consiste?

es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

pregunta 7:que es una etiqueta, funciona en mayuscula y en minuscula?

En HTML una etiqueta es usada para crear un elemento, da igual si lo escribimos en mayuscula o minuscula

pregunta 8:a que llamamos elemento html, cual es su anatomia? explicar sus partes

Un elemento es parte de una página web.HTML, un elemento puede contener un elemento de datos o un fragmento de texto o una imagen, o tal vez nada, su anatomia se componde de: su inicio, un atributo y su valor o contenido de texto adjunto, y su cierre

pregunta 9:que es un atributo? dar 2 ejemplos

Un atributo amplía una etiqueta (tag), cambiando su comportamiento o proporcionando metadatos. Un atributo tiene la forma nombre=valor(charset, content).

pregunta10:que hace la etiqueta P,Y,strong,emm,sub,sup,u,br

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.

el elemento I Muestra el texto marcado con un estilo en cursiva o italica. hola

El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto. hola

el elemento em es apropiado para marcar con énfasis las partes importantes de un texto. hola

el elemento Sup define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice. hola

el elemento sub define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice. hola

el elemento U Muestra el texto subrayado. hola

el elemento Br produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significante.

cite: Marca una referencia a una fuente, o el autor de un texto citado, ejemplo: Hola

hr: representa un cambio de tema entre párrafos, ejemplo:

abbr: representa una abreviación o acrónimo, ejemplo: Hola

code: Es el apropiado para marcar el código de un programa. Ejemplo: Hola

span: Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.
